Для работы сервера нужно:
- папка build с файлами сервера (собирается gulp'ом из папки src)
- зависимости (папка node_modules)
- файл package.json (если dependencies не установлены глобально на машине)

Скачать зависимости можно с помощью файла package.json:
- раздаем интернет через телефон (через корпоративный интернет скачать не получится)
- создаем временную папку, кладем в нее файл package.json
- открываем терминал, переходим в эту папку
- скачиваем зависимости (папка node_modules) командой npm i
- переносим скачанные зависимости (папка node_modules) в папку webserver

Перезапуск сервера:
- в диспетчере задач убиваем процессы Node.js: Server-side JavaScript
- обновляем файлы сервера (папка build) и зависимости (папка node_modules), если необходимо
- запускаем runServer.cmd (через планировщик заданий)
